# Customer Insights Analysis Project

## Problem Statement
The project aims to analyze customer behavior, sales performance, and business trends to generate meaningful insights for better decision-making.

## Tools Used
- Jupyter notebook for python
- MySQL
- Power BI

## Steps Performed
1. Loaded the dataset into Python for data cleaning and preprocessing
2. Performed exploratory data analysis using Pandas and NumPy
3. Imported the dataset into SQL and wrote analytical business queries
4. Connected the processed data to Power BI
5. Built interactive dashboards and visualizations
6. Extracted meaningful business insights from the dashboard

## Key Analysis Performed
- Customer behavior analysis
- Sales and profit analysis
- Regional performance analysis
- Product/category performance analysis
- Trend and pattern identification
- Business KPI tracking

## Conclusion
Analyzed customer and sales data using Python, SQL, and Power BI to identify business trends, customer insights, and performance metrics through interactive dashboards and analytical queries.
